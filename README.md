# CGVYAPAM Assistant Draftsman MTProto Quiz Bot

Advanced Pyrogram (MTProto) quiz bot built for CGVYAPAM Assistant Draftsman exam preparation.

**Features**
- All question types from the official pattern (A&R with all variants, Match the Following, Arrange in Sequence, Select Correct Statements / Multi-select, Numerical, Image support)
- Bulk upload 100-150 questions via single JSON file
- Full admin panel (only for OWNER_ID)
- MongoDB storage
- Render-ready

## Deployment
1. Deploy on Render (or Railway/Heroku)
2. Add the 5 environment variables
3. Send /start to your bot
